# Shared To-Do List Full Stack Web App

## Technologies
- Frontend 
  - HTML5
  - CSS
  - JavaScript
- Backend
  - Microservice - data service
  - Microservice - authentication & authorization service
  - Database - MySQL
- Proxy / Load Balancer
  - NGINX
- Deployment
  - docker


## Quickstart
### Development
`docker-compose -f docker-compose-dev.yaml up -d`
